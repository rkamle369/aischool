# AI Quiz

Mobile-first, voice-enabled quiz app with course and level progression.

## Features

- Course picker with multiple domains.
- Level-based flow (10 questions per level).
- Touch and voice answer modes (`Option A/B/C/D`).
- Instant correctness + short explanation.
- Next level unlocks only when score is >= 90%.
- LiveKit connection setup reusing backend token flow.

## Run locally

```bash
npm install
npm run dev
```
